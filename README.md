# Tello-Drone

A project for controlling a DJI Tello drone using a python interface using manual flight and video streaming - with the potential for expanding to autonomous flight.

## Table of Contents
- [Project Overview](#overview)
- [Features](#features)
- [Prerequisites](#prerequisites)
- [Usage](#usage)

## Project Overview
This porject provides Python scripts to communicate and control a DJI Tello drone. Handles basic commands with the keyboard and interface like takeoff, land, movement, and even flips!

## Features
- Basic drone commands: takeoff, land, forward/back/left/right/up/down
- Clean interface with video feed feature

## Prerequisites
- DJI Tello drone

## Usage
- python pygame_test.py
